# Test Plan – Netflix

**Test Plan ID:** TP-NFLX-001  
**Project Name:** Netflix – Search Functionality  
**Document Version:** 1.0  
**Date:** 2025-08-10  
**Prepared by:** Gonzalo Bessone  

## 1. Objective  
Verify that the Netflix search feature returns accurate results for valid titles, and displays appropriate messages for invalid or non-existing titles.  

## 2. Scope  
**Included:**  
- Search with valid movie/series titles.  
- Search with invalid or non-existing titles.  

**Excluded:**  
- Voice search functionality.  
- Search filtering by genre or category.  

## 3. Test Strategy  
Manual functional testing will be performed on the Netflix web platform using Chrome and Firefox in their latest stable versions.  

## 4. Entry Criteria (Definition of Ready)  
- Access to Netflix with a valid test account.  
- Browsers updated to the latest stable version.  
- Test data prepared.  

## 5. Exit Criteria (Definition of Done)  
- All planned test cases executed.  
- No high or critical severity defects remain open.  
- Final report delivered.  

## 6. Resources Required  
- 1 Manual QA Tester.  
- Computer with internet access.  
- Documentation tool (Google Sheets, Excel, or Jira).  

## 7. Estimated Schedule  
- Test case design: 1 day.  
- Execution: 1 day.  
- Final report: 0.5 days.
